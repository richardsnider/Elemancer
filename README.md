# Elemancer #

Elemancer is a javascript framework for creating/controlling custom html elements. It is only dependent on Vanilla JS.

### What is this repository for? ###

* This repository is for hosting the source code.

### How do I get set up? ###

* Download source.
* npm install
* nodejs server.js
